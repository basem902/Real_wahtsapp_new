## 1. نظرة عامة على المشروع

### فكرة المشروع
منصة SaaS متكاملة لإدارة العقارات في سوق جدة، تدمج بين:
- **بوت واتساب ذكي** للتواصل التلقائي مع العملاء
- **لوحة تحكم احترافية** لإدارة العقارات والمسوقين والصفقات
- **نظام توزيع ذكي** للمسوقين العقاريين

### مكونات المشروع الحالية
| المكون | المسار | الوصف |
|--------|--------|-------|
| smart-realestate-agent | `/smart-realestate-agent/` | المنصة الرئيسية - Next.js 14 + Supabase + OpenAI |
| Whats_Real Backend | `/Whats_Real/backend/` | الخادم الخلفي - Express.js + Wasender API |
| Whats_Real Dashboard | `/Whats_Real/dashboard/` | لوحة التحكم - Next.js + Tailwind |

### التقنيات المستخدمة
| الطبقة | التقنية |
|--------|---------|
| Runtime | Node.js 18+ |
| Backend Framework | Express.js 4.18 |
| Frontend Framework | Next.js 14 (App Router) + React 18 + TypeScript |
| Database | PostgreSQL 16 (Supabase) |
| Cache | Redis 7 (اختياري) / In-Memory |
| Authentication | JWT + Bcrypt + TOTP |
| Encryption | AES-256-GCM + HMAC + SHA256 |
| File Storage | Supabase Storage (S3-compatible) |
| Messaging API | Wasender (WhatsApp Business) |
| Styling | Tailwind CSS 3 + shadcn/ui |
| Charts | Recharts + Chart.js |

## 9. البنية التقنية

### 9.1 هيكل المشروع المقترح (بعد التوحيد)
```
aqari-jeddah/
  apps/
    backend/                  ← Express.js API Server
      src/
        routes/              ← API endpoints
        services/            ← Business logic
        middleware/           ← Auth, Rate Limit, etc.
        bot/                 ← WhatsApp bot logic
        integrations/        ← Wasender, OpenAI, FCM
        utils/               ← Helpers
      tests/
      server.js

    dashboard/               ← Next.js Admin Dashboard
      app/                   ← App Router pages
      components/            ← UI components
      lib/                   ← Utilities
      hooks/                 ← Custom React hooks

    mobile/                  ← React Native App (مستقبلي)

  packages/
    shared/                  ← Shared code
      types/                 ← TypeScript types
      constants/             ← Shared constants
      utils/                 ← Common utilities

  database/
    migrations/              ← SQL migrations
    seeds/                   ← Initial data
    schema.sql              ← Full schema

  docs/                      ← Documentation
  infrastructure/            ← Docker, CI/CD

  package.json              ← Monorepo root
  turbo.json                ← Turborepo config
  .env.example
  .gitignore
```
