## 1. نظرة عامة على المشروع

### ما هو المشروع؟
**عقاري جدة** هي منصة SaaS متكاملة لإدارة المبيعات العقارية عبر واتساب بالذكاء الاصطناعي. تستهدف الشركات العقارية والمسوقين في جدة والمملكة العربية السعودية.

### الهدف الرئيسي
تحويل رسائل واتساب إلى صفقات عقارية مغلقة من خلال:
- بوت ذكي يرد تلقائياً على استفسارات العملاء بالعربية
- تقييم جدية العملاء وتوزيعهم على المسوقين
- لوحة تحكم شاملة للإدارة والتحليلات
- نظام اشتراكات متعدد المستأجرين (Multi-tenant)

### هيكل المشروع العام
```
Real_wahtsapp_new/
├── landing-page/              → صفحة الهبوط التسويقية (HTML/CSS/JS)
├── smart-realestate-agent/    → التطبيق الرئيسي (Next.js 14 + Supabase)
├── plan/                      → خطط المشروع والتنفيذ
├── .claude/                   → إعدادات التطوير
└── info/                      → تقارير المشروع (هذا الملف)
```

#### نظام الذكاء الاصطناعي (6 وكلاء):

| **Qualification Agent** | تجميع بيانات العميل وتأهيله |

Viewer (المشاهد)   → قراءة فقط

### 2.4 الأمان والحماية

#### الممارسات الجيدة:
- Rate Limiting (5 محاولات تسجيل/دقيقة، 1000 webhook/دقيقة)
- Prompt Injection Prevention (تنظيف المدخلات)
- Bot-to-Bot Loop Prevention
- Webhook Signature Verification (HMAC)
- Security Headers (X-Frame-Options, X-Content-Type-Options, X-XSS-Protection)
- Secret Scanner Script (فحص آلي للأسرار المضمنة)
- Zod Validation على جميع المدخلات
- Encryption للمفاتيح الحساسة

#### مشاكل أمنية محتملة:
1. **INTERNAL_API_KEY** قد يكون فارغاً بشكل افتراضي
2. **CI/CD** يستخدم placeholders بدلاً من GitHub Secrets
3. **بعض الأخطاء** قد تكشف معلومات حساسة في الـ Logs
